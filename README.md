- ### 변경사항

1. #### UX

   - 도시별 날씨 상태의 종류가 세분화되어 있어 사용자가 직관적으로 받아들이기 힘들었습니다.

   - 이를 비슷한 날씨를 하나의 그룹으로 묶었습니다.
   - 이로써 정보 전달의 효용성이 증가하였습니다.

2. #### UI

   - 날씨상태를 텍스트에서 아이콘으로  변경함으로서 직관적으로 받아들일 수 있도록 하였습니다.
   - 다소 심심한 날씨정보에 활력을 돋우기 위해서 '기상캐스터 잔나'를 넣어주었습니다.
